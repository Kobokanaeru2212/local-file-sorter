# Local File Sorter Automation

Script Python sederhana untuk merapikan file di dalam sebuah direktori secara otomatis berdasarkan ekstensi filenya (Dokumen, Gambar, Installer, dll) ke dalam folder kategori yang terstruktur.

## Fitur
- Otomatis mendeteksi ekstensi file dan mengelompokkannya.
- Proteksi script agar tidak memindahkan dirinya sendiri saat dijalankan.
- Mendukung argumen baris perintah (CLI) untuk target folder yang spesifik.

## Cara Penggunaan
Taruh file `main.py` di dalam folder yang berantakan, lalu jalankan melalui Terminal / Command Prompt:
```bash
python main.py
